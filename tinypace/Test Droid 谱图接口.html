Controller 根路径: ip<:port>/testdroid/v2/graph/path

# 查询方法

# 默认查询

含义:

根据项目详情 id 查询是否有相应的图谱, 如果没有则新建一个 init 根节点, 并返回, 若没有对应的 APP 则提示错误. 如果有相应的图谱, 则直接返回只有场景的图谱

路径:

/testdroid/v2/graph/path?projectDetailId={projectDetailId}

请求参数:

projectDetailId: 对应的项目详情 id
1

请求方式:

GET

返回值:

正确

{
    "code": 200,
    "errorMsg": "success",
    "data": {
        "nodes": [
            {
                "name": "登录遇到问题",

# 添加一个游离节点

/testdroid/v2/graph/node

{
    "name":"爱奇艺", //节点名称
