# algo-api 使用文档

# 算法接口

请求地址: ip:port/algo 请求方式: POST

请求参数:

algoParameter : 前端生成的 json 格式的房间数据

请求头:

access_token: 客户调用时的凭证(测试时使用"abcd")

Content-Type: application/json (数据格式)

返回数据:

code: 请求状态码(自定义)

message: 解释信息/错误信息

data: 请求返回数据

algoCode: 请求记录 id (已加密)

algoResult: 算法返回值

algoRequestFile: 算法请求数据生成的文件的 id (已加密)

algoResponseFile: 算法返回参数生成的文件的 id (已加密)

# 文件生成接口

请求地址: ip:port/algo/inventory?code=xxx

请求方式: get

请求参数:

code: 算法接口返回的 algoCode

请求头:

access_token: 客户调用时的凭证(测试时使用"abcd")

Content-Type: application/json (数据格式)

返回数据:

code: 请求状态码(自定义)

message: 解释信息/错误信息

data: 请求返回数据 -- -- 生成的 Excel 文件的 id (已加密)
